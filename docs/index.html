Spatiotemporal Colorado Potato Beetle (CPB) Modeling

This project investigates the spatiotemporal dynamics of Colorado Potato Beetle (CPB) populations in Wisconsin. We integrate field data, climate data, and crop data to build predictive models for CPB abundance and insecticide resistance risk.

Project Pipeline

  1. Data Extraction and Preprocessing (Python)
    • Extract CPB abundance from raw data (01_extract_clean_combine.ipynb)
    • Add climate variables and extreme weather indicators (02_add_climate_data.ipynb)
    • Remove outliers and multicollinearity, finalize dataset (03_outlier_multicollinearity_final_cleaning.ipynb)
  2. Crop and Potato Intensity Features (R)
    • Calculate crop type from CDL (majority method)
    • Compute potato proportion (weighted & classical)
    • Compute potato intensity (weighted & classical with buffer zone)
    • Script: 01_crop_intensity_proportion.Rmd
  3. Final Dataset
    • Processed dataset: data/processed/final_data_for_modeling.csv
  4. Modeling (R)
    • Mixed Model (lmer)
    • GAMM Model
    • spaMM Model
  5. EDA and Visualization (Python)
    • EDA and Visualizations

Notes

  • CDL raster files: Large files stored in data/cdl/. Not uploaded to GitHub due to size limitations.
  • Intermediate tables in data/interim/ were generated by both R and Python.
  • Final dataset for modeling is data/processed/final_data_for_modeling.csv.
